# Intro to Data Science - Group Assignment

## How to run the data cleaning script

1. Download all files from this repository
2. Open `merge_clean_ids_project.R` in RStudio
3. In RStudio menu: **Session > Set Working Directory > To Source File Location**
4. Run the script
5. The cleaned dataset will be saved as `master_dataset.csv`

## Files in this repository

- `merge_clean_ids_project.R` - Data cleaning and merging script
- `gdp-per-capita-worldbank.csv` - GDP per capita data
- `youth-not-in-education-employment-training.csv` - NEET data
- `continents-according-to-our-world-in-data.csv` - Continent classification
- `government_expenditure_on_education.csv` - Education expenditure data
